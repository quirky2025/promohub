## 0. 全链路

```
客户填 /supply-chain/quote  →  sourcing_requests(收件箱)
        →  [断档要建的部分] ↓
        →  录入多家工厂 EXW 报价  →  计价工具(工厂比价 + 运费比价)
        →  选最优工厂 + 运费 → 出客户 AUD 报价(冻结快照)
        →  同一批工厂产品可「上架 INDENT」给前台目录
```

现状:`/supply-chain/quote`(前台 3 步表单)和 `sourcing_requests` 后台收件箱**已存在**;收件箱目前只能看/回/记笔记。本规格补后面全部。

## 1. 决策锁定表(Lily 已确认)

| # | 决策 | 结论 |
|---|---|---|
| 1 | 工厂→货代内陆运费 | **每单手动填**,不存固定值 |
| 2 | 汇率 RMB→AUD | **全局一个、手动改**;改后 INDENT 价自动重算;报价时用当时汇率并**冻结进快照** |
| 3 | 本地费用(AUD) | **每单不同**,一般中国全包(≈0),**字段保留**备用 |
| 4 | MARGIN | **一次性报价**=每单 Lily 自填;**INDENT 目录产品**=存默认 margin(默认 1.40,逐个可改) |
| 5 | FOB | **不要 FOB,只用 EXW**(FOB 价不准) |
| 6 | 工厂 vs Suppliers | **分两张表**(工厂=海外 RMB+产品+计价;suppliers=本地备货) |
| 7 | 多工厂比价 | 一条请求可录**多家工厂 EXW**,工具算到岸 AUD **并排比**,选最省 |
| 8 | 国际运费 | 货代费率引擎(见 §4),Express **自动选最便宜**承运商 |
| 9 | 性质 | 全程**估算**(费率表"出货前单询"),最终以货代确认为准 |

## 2. 计价公式(每件 AUD)

```
中国方(RMB)：
  EXW 单价(按数量分档,工厂报)
+ 工厂→货代内陆运费(总额 ÷ 数量,每单手填)
+ 国际运费(运费引擎算出总额 ÷ 数量)
= 中国方小计 RMB/件
× 汇率(RMB→AUD,全局手动)
= 中国方 AUD/件

本地方(AUD)：
+ 本地费用(总额 ÷ 数量,每单填,常为 0)
= 到岸成本 AUD/件
× MARGIN(目录=默认1.40可改;报价=每单自填)
= 客户单价 AUD/件(excl. GST)
```

多工厂时:每家工厂各算一条到岸成本 → 并排比 → 选。
运费多方式时:每方式各算一条 → 并排比(默认最省)→ 选。

## 3. 数据模型

### 3.1 factories(工厂)
`id, name, contact_name, email, phone, wechat, country(default China), currency(default RMB), payment_terms, notes, created_at`

### 3.2 factory_products(工厂产品)
`id, factory_id FK, name, factory_sku, category, subcategory, description, images[],`
`carton_length_cm, carton_width_cm, carton_height_cm, units_per_carton, unit_weight_kg,`  ← 算计费重用
`goods_class(enum: class1|class2|class3|special|high_value),`  ← 算附加费用
`default_margin(default 1.40),`
`publish_to_indent(bool), public_product_id FK→products(上架后关联), created_at`

### 3.3 factory_product_prices(工厂 EXW 分档价)
`id, factory_product_id FK, min_qty, exw_unit_rmb`  (只存 EXW,不存 FOB)

### 3.4 settings(全局)
`fx_rate_rmb_to_aud(手动改), fx_rate_updated_at, default_margin`

### 3.5 sourcing_quotes(报价 / 多工厂比价)
挂在 `sourcing_requests` 上;一条请求可有多条工厂候选:
`id, sourcing_request_id FK, factory_product_id FK(或自由填), qty,`
`exw_unit_rmb, inland_freight_rmb_total(手填), freight_snapshot_id FK(选定的运费),`
`local_fees_aud_total(手填), fx_rate_used, margin_used,`
`landed_unit_aud, customer_unit_aud, is_chosen(bool), created_at`

## 4. 国际运费引擎(基于货代真实费率表)

### 4.1 三维身份:origin + carrier + service
前端**强制两步选**:先 origin,再只显示该 origin 有费率表的 carrier。香港DHL ≠ 大陆FedEx,绝不混。

### 4.3 两种计费模型(rate_rows.pricing_model)
- `first_additional`:首重价 + ⌈(计费重 − 首重单位)/续重单位⌉ × 续重价(低重量段用)
- `flat_per_kg`:计费重 × kg单价(过阈值后:空派 21kg+、海派 11kg+)

### 4.4 计费重
`体积重 = 长×宽×高(cm) ÷ 6000`(抛货比 1:167);`计费重 = max(实重, 体积重)`,再**向上取整**(空派 0.5kg、海派 1kg — 待 Lily 最终确认进位粒度),并套最低计费重(澳邮单边>100cm 时最低 22kg)。**重量档左闭右开** `from ≤ kg < to`。

### 4.5 品类附加费(按 goods_class,加在基础运费上)
`一类 ¥1/kg(纺织/钳/灯具…) · 二类 ¥2/kg(餐具/包/杯/鞋/玩具…) · 三类 ¥3/kg(电器/蓝牙…) · 特殊 ¥3/件(U盘/手表/PCB/IC卡) · 高价值 单询(手填)`

### 4.6 额外费(规则化;确定的自动加,实报实销的标记手填)
- 关税/清关:**仅"不含税"渠道**适用 —— 申报>1000AUD → 120AUD/票清关 + 货值15% DUTY。**空派/海派已含税** → 每张费率表一个 `tax_inclusive` 开关(空派海派默认 true,Express 默认 false,可改)。
- 保险:货值 0.3%,低消 ¥50/票(可选)。
- 超重:单件计费重 ≥30/50/80kg → ¥100/150/300/件。
- 超长:单边 ≥110…600+cm → ¥100…2250/件(分档)。**超长+超重叠加收**。
- 实报实销(熏蒸/商检/改单500AUD/二次派送/磁检500/报关350…)→ 标记为手填行。

### 4.7 不可派送邮编(黑名单,先拦截)
`2059, 2898-2899, 6798-6799, 7151-7154, 6324, 4280, 4300, 3336, 4875` → 返回"无服务/单询",不出数字。

### 4.8 版本化 + 快照(可改价 + 可复算)
- 费率表**只增不改**:改价=新版本,旧版本保留;每个 (origin,carrier,service) 同时只一个 active。
- 报价/下单时把整笔计算(用到的费率行 + 汇率 + 明细 JSON + estimate_flag)**冻结成不可变快照**,以后改价不影响旧单。
- 前端估算标注:`基于费率表 v{版本}, 生效 {日期};费率常变,最终以下单时确认为准`。

## 5. 计价工具(UI)

输入:工厂产品(或自由填)+ 数量 + 邮编 + 箱规/重量(目录产品自动带出)。
输出:
1. 计费重自动算。
2. **运费方式并排比**(Express 自动最省 / 空派 / 海派 — 价 + 时效 + /件),你点选。
3. **多工厂并排比**(各家 EXW → 到岸 AUD/件),你点选。
4. 折进:EXW + 内陆运费(手填)+ 选定运费 + 本地费(手填)+ 汇率 + margin(自填)→ **客户单价 + 总价**。
5. 生成报价(冻结快照)→ 回客户;或「上架 INDENT」。

---

## 6. 上架 INDENT

工厂产品打开 `publish_to_indent` → 用「默认 margin」按各数量档算出 AUD 分档价 → 写入/关联前台 `products`(indent 类型),出现在 `/indent` 目录,展示与现有 indent 产品一致。汇率改动 → 这些价自动重算。

## 7. 建设阶段(分模块建)

| 阶段 | 内容 | 验收 |
|---|---|---|
| **P1 运费数据** | 6 张费率表 + 真实数字 + `tax_inclusive`/版本字段;可改价编辑页 | 改价存新版本、旧版本仍在 |
| **P2 运费引擎** | 纯计算:定表→计费重→档→模型→附加费→额外费→汇率→明细;Express 自动最省 | 数字对得上货代发票(尤其档边界) |
| **P3 快照** | 报价/下单冻结快照 + estimate_flag | 改价后旧报价原样复算 |
| **P4 工厂 + 工厂产品** | factories / factory_products / EXW 分档价;箱规/单重/品类/默认margin 字段 | 建工厂→建产品→录 EXW 档 |
| **P5 计价工具** | 多工厂比价 + 运费比价 + 折算客户价 + 生成报价(挂 sourcing_request) | 例:100布袋出客户价,换运费即时变 |
| **P6 上架 INDENT** | publish_to_indent → 自动算 AUD 分档价 → 前台目录;汇率改动自动重算 | 工厂产品出现在 /indent |

运费(P1–P3)先行,可独立试算;再工厂/计价(P4–P5);最后上架(P6)。
