# QuirkyPromo 增长全貌一页纸 · 2026-07

> 逻辑:每一层是上一层的地基,跳层做 = 钱和力气漏掉。**现在位置:第 0 层收尾,第 1 层开打。**

## 第 0 层 · 技术地基(✅ 基本收工,7 月第一周完成)
GSC 监控 / GA4 六事件 + 维度 / 动态 sitemap / 945 个 404 修复 / FAQ 页 + schema / AS meta 模板升级(进行中)/ 每周看板。
**验收标准**:收录数稳步上爬、埋点数据在流。剩的都是尾巴。

## 第 1 层 · 存量做深(🔶 现在开打,7–8 月主战场)
- **类目/子类目页深化**(= 最迫切的 LP 文案):意图段 + 选购指南 + 印刷说明 + 品类 FAQ + 内链。一周 3–5 页,从供给最深的品类开始。
- **产品页 SEO 精细化 = 改模板,不逐页写**:6000 页的精细做在模板和字段层——meta 模板肥化(已交数据线)、JSON-LD(已有)、specs 完整度、图片 alt、Similar products 内链、产品页 FAQ 块。等 GSC 数据成熟(3–6 个月)再挑 top 20 产品逐页手工精修。
- 首批颜色集合页(black tote 等 2–3 个)。
**验收标准**:深化页的目标词进前 20、类目→产品点击率上升。

## 第 2 层 · 增量扩张(8–10 月,与第 1 层交叠推进)
- 新落地页:场景(Trade Show / Christmas)、行业(Hospitality / Tradie)、Teamwear 拆分(见 LANDING_PAGE_MAP.csv)
- **Blog 内容**(不是随笔,是带询盘的决策类文章,内链回产品)
- **5 条视频**(How to Order / Proof 优先)
- 评价体系 + 企业 logo 墙(解锁 Review schema + 消掉 GSC 提示)
- 留资弹窗 + 首单券(越早越好,可提前)
- 询盘流程:自动确认 + 3 工作小时承诺兑现 + follow-up 模板
**验收标准**:enquiry 数与来源清晰、quote→order 转化率有数。

## 第 3 层 · 放大器(10 月+,前两层的数据说了算)
- **Merchant Center** → **Google Ads / Shopping**(⚠ 烧钱前必须知道 enquiry→order 转化率和客单,否则是往漏的桶里倒水)
- 社交媒体(有案例/评价/视频素材后才有内容可发)
- Google Business Profile / GEO 精调 / Email 季节性群发规模化
**解锁条件**:第 1 层排名见效 + 第 2 层转化数据成型 + 有评价背书。

## 贯穿全程(每周,不分层)
周一 GSC 例行 + 看板 / 询盘当天跟进 / 复购跟单。

## 三个原则
1. **顺序不能跳**:没有深页面,广告点进来也不转化;没有转化数据,广告出价就是瞎猜。
2. **存量优先于增量**:已收录页面做深,比新建页面见效快。
3. **每层用数据验收**,不凭感觉进入下一层。
