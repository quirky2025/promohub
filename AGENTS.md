# 前端风格规范（必读 / MUST READ before any frontend work）

改任何**前台页面**（客户能看到的页面）前，先读 **`FRONTEND_STYLE_GUIDE.md`** 并照它做。
要点：浅底一律纯白 `#fff` + 黑字 `#1a1a1a`（不准灰底/灰字）；深色用 navy `#1B2A4A` + 白字；
**灰字禁令适用于一切文字**，包括辅助小字、时间戳、表头、占位提示——一律黑 `#1a1a1a` 或 navy，不得用浅灰（Lily 2026-07-21：灰的看不清）；
表格标题栏/折叠条 = navy 居中白字；同类页面 hero 高度统一；"浏览产品"链到 `/promotional-products`（站上没有 `/category`、`/products`、`/catalog` 总页，链了会 404）。
全站风格有改动时，同步更新 `FRONTEND_STYLE_GUIDE.md`。
