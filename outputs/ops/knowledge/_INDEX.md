# 运营知识库索引 · 存储即 promohub/outputs/ops/knowledge/

> 用途:给落地页/blog/FAQ/GEO 文案供弹药。Lily 投料(拍照/摘要丢给运营 chat),运营 chat 蒸馏入库。
> 铁律:①杂志原文不进公开文案,入库的是事实和洞察,出文案时重写;②数字必须带来源+年份;③过期(>3 年)数据用前先核。
> 双语规则:**英文 = 事实原话/文案钩子(直接可用于英文文案),中文 = 解读和用途说明**。

## 目录
- **scenarios/** 使用场景 —— 展会、企业送礼、员工入职、活动会议、季节性(EOFY/圣诞)
  → 喂:场景落地页(L4)、选品灵感类 blog
- **craft/** 工艺与材质 —— 印刷方式、面料材质、artwork 文件知识
  → 喂:决策类 blog(print methods 对比)、品类选购指南、FAQ
- **industry/** 行业数据 —— 促销品 ROI/留存率统计、市场趋势(eco/科技品)、品牌供应商动态
  → 喂:文案钩子数字、About/信任内容、GEO 可抽取事实
- **buyers/** 买家行业 —— hospitality、建筑 tradie、地产、教育…各行业买什么怎么买
  → 喂:行业落地页(L5)

## 条目格式(每条 3–5 行,写在对应主题文件里)
```
### [一句话标题]
来源:XX 杂志/网站,2026-07
事实:……(数字带单位和年份)
可用于:/trade-show-giveaways-australia 落地页 / blog《XX》
备注:(可选)原文关键措辞、待核实点
```
